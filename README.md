# AI-Powered-Energy-Consumption-Forecasting
"Using Neural Networks to optimize energy usage for Smart Cities."

## 🎯 Project Objective

Predicting electricity demand to reduce wastage and prevent blackouts. This system helps power grids balance supply and demand in real-time.
## 🛠️ Tech Stack

    AI Model: Multi-Layer Perceptron (MLP) Regressor.

    Deployment: Flask API for real-time forecasting.

    Data: Smart Grid consumption logs.

## 🚀 Results

    Functionality: Successfully trained and deployed an AI model that takes 'Hour' and 'Day' inputs to forecast power load.

    Impact: Aims to reduce the 70% power loss caused by poor forecasting in developing smart grids.
